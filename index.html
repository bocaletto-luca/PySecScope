PySecScope

About PySecScope

PySecScope is a comprehensive Linux enumeration tool featuring a clean and responsive GUI built with Tkinter. The tool gathers system, network, service, and security data, and exports reports in multiple formats (Text, JSON, CSV).

Features

  • Modular scan modules for system, user, network, and services.
  • Thorough mode for additional security checks (SUID/SGID files, cron jobs, sudoers, logs, etc.).
  • Multi-format report export (Text, JSON, CSV).
  • Real-time progress bar and status updates.
  • Scan cancellation option.
  • Configurable defaults via an INI file.
  • Robust logging for troubleshooting (saved in pysecscope.log).
  • Privilege verification with a warning if not run as root.

Download
