PySecScope

A Comprehensive Linux Enumeration Tool with a Tkinter GUI

View on GitHub

Features

  • Modular Scan Modules: Collects system, user, network, and service information.
  • Thorough Mode: Additional checks for SUID/SGID files, file capabilities, cron jobs, sudoers, log files, firewall rules, and more.
  • Multi-Format Reporting: Export reports in Text, JSON, or CSV formats.
  • Real-Time Progress: Responsive GUI with a progress bar and status updates during scanning.
  • Cancel Scan Option: Ability to cancel an ongoing scan.
  • Configuration: Customizable defaults via a configuration file (pysecscope.ini).
  • Comprehensive Logging: Detailed logging via the Python logging module (saved in pysecscope.log).
